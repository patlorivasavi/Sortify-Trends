Sortify Trends 🚀

Real-Time Social Media Trend Ranking & Algorithm Visualizer

Sortify Trends is an interactive Algorithm Visualization Platform that demonstrates how sorting, searching, and greedy algorithms power real-time social media trend ranking systems.

The platform simulates how trending posts and hashtags are ranked based on engagement metrics like likes, shares, comments, and relevance scores.

This project is developed as part of a Design and Analysis of Algorithms (DAA) Project-Based Learning (PBL) to demonstrate practical applications of algorithms in modern digital platforms.

📌 Features
🔹 Algorithm Visualizations
Interactive visual demonstrations of popular algorithms:
Merge Sort,Quick Sort,Heap Sort,Binary Search,Greedy Optimization

🔹 Sorting Visualizer
Watch how sorting algorithms organize trending posts in real-time with animated bars and step controls.

🔹 Binary Search Visualizer
Step-by-step visualization of searching a specific trend in sorted datasets.

🔹 Heap Priority Queue
Visual representation of a Max Heap Tree to show how top trending hashtags are prioritized.

🔹 Algorithm Comparison
Compare multiple algorithms side-by-side based on:
Time Complexity,Execution Speed,Number of Operations

🔹 Engagement Simulator
Simulates how engagement factors affect trend ranking using weighted scoring.

🔹 Multilingual Feed
Sample posts are displayed in multiple languages including:Telugu,Kannada,Hindi,Tamil,Korean


English

This demonstrates how global platforms handle multilingual social media content.
